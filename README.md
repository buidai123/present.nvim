# present.nvim

This is a markdown presentation for neovim

# Feature

Can execute the codeblocks in current slide

```lua
print("Hello World!")
```

# Usage

for lazy.nvim

```lua
return {
    "buidai123/present",
    config = function()
        require("present")
    end
}
```

use `n` and `p` to navigate around slides

`q` for quite the presentation
